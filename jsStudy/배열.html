배열

자바스크립트에서 변수 선언시 별도의 자료형 지정이 없기 때문에 어떤 자료형의 값들이던 전부 하나의 배열공간에 담을 수 있다. (자바 컬렉션과 유사)

배열 선언시 배열의 크기를 지정한 채로 선언하거나, 크기를 지정하지 않아도 된다. (크기는 제약이 없다.)

Array 객체의 메소드

1 indexof(찾고자하는요소) : 배열에서 해당 요소가 위치해있는 인덱스를 반환

2 concat(배열, 배열, ...) : 여러개의 배열을 합치고자 할때 사용

3 reverse(): 배열에 담긴 값들을 역순으로 바꿔주는 메소드

배열 반복문

배열을 통한 반복작업(forEach)

배열 탐색(find, filter)

객체로 이루어진 배열에서 내가 원하는 값만 찾을 때 사용

배열 변형(map)

현재 배열을 바탕으로 각 요소에 함수를 호출하여 변경된 결과값을 반환함
